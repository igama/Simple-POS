Demo content

To reproduce our demostration site you need to create some extra blocks, stories and pages in your installation. To make your life easier we provide here everything you may need for this. Blue Masters comes with predefined CSS rules for your content.

Note: You don't need to take extra care about references to images. Images for demo content are included into theme's distribution.

Prerequisites

As the Demo content of Corporate Clean installation requires the configuration of PHP input format at several points, please make sure you have activated the PHP filter module right after installing.

Slideshow

Corporate Clean theme comes with a slide show implementation hard coded into the page.tpl.php. This choice allows you have a slideshow up and run just after the theme installation. Then you could dig into the source code and modify it according to your needs.

To learn more about the Slideshow scroll down to the Advances Topics of this wiki page.

Get rid of the hardcoded slideshow

Corporate Clean gives the option to disable the build in, hardocode, slideshow implementation without dealing with PHP at all. Simple goto Corporate Clean theme settings and disable it.

Some theory about the slideshow markup

Corporate Clean is configured to understand the existance of a slide show even this is hardcoded into the page.tpl.php, comes with a custom html block or produced be a sophisticated View based mechanism. The only rule is that the slideshow markup needs to be equivalent with the following markup. 

 <?php if (theme_get_setting('slideshow_display','corporateclean')): ?>
 <?php if ($is_front): ?>
 
    <!--slideshow-->
    <div id="slideshow">
 
        <!--FIRST SLIDE:slider-item-->
        <div class="slider-item">
            <div class="content">
 
                <!--slider-item content-->
		  YOUR HTML CODE
                <!--EOF:slider-item content-->
 
            </div>
        </div>
        <!--EOF:slider-item-->
 
 
        <!--SECOND SLIDE:slider-item-->
        <div class="slider-item">
            <div class="content">
 
                <!--slider-item content-->
		  YOUR HTML CODE
                <!--EOF:slider-item content-->
 
            </div>
        </div>
        <!--EOF:slider-item-->
 
 
    </div>
    <!--EOF:slideshow-->
 
    <!--slider-controls-wrapper-->
    <div id="slider-controls-wrapper">
        <div id="slider-controls">
            <ul id="slider-navigation">
                <li><a href="#"></a></li>
                <li><a href="#"></a></li>
            </ul>
        </div>
    </div>
    <!--EOF:slider-controls-wrapper-->
 
<?php endif; ?>  
<?php endif; ?>

Every slider-item at the code above constitutes one slide of the slide show. Therefore, in order to add a new slide, you need to add the following code: 

        <!--slider-item-->
        <div class="slider-item">
            <div class="content">
 
                <!--slider-item content-->
		  YOUR HTML CODE HERE
                <!--EOF:slider-item content-->
 
            </div>
        </div>
        <!--EOF:slider-item-->

You could place your content where it says "YOUR HTML CODE HERE".

For every slider-item that you will add, you need to add a list-item of the slider-controls-wrapper as well: 

<ul id="slider-navigation">
      <li><a href="#"></a></li>
      <li><a href="#"></a></li>
      ...
</ul>
